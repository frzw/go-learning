# 三大问题
1. go是什么？
    - 快速的、静态类型的编译型语言，简洁高效模块化，天生并发，垃圾回收机制、运行时反射机制。
        - 并发：不要通过共享内存来通信，而要通过通信来共享内存。
            - goroutine
            - channel
            - select
            - sync.Mutex
        - 垃圾回收：
            - 简单却高效的标记-清除回收器。
            - 高效、低延迟的并发回收器。
            - 目前 gccgo 还没有回收器，同时适用 gc 和 gccgo 的新回收器正在研发中。
        - 反射机制：
            - 反射从接口值到反射对象。
            - 反射从反射对象到接口值。
            - 要修改反射对象，该值必须可设置。
2. go能做什么？
    - web开发
    - 微服务
    - 区块链
    - 爬虫
3. go怎么使用？
    - 下载对应系统的go
    - 设置工作环境变量
    - 测试
